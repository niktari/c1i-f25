  • Harmonic Collection: Entry #10

    Create an index.html file, style.css file, and script.js file in your entry-10 folder. Make sure you link your script.js file in your index.html file by using the <script></script> tag!

    Practice using inputs by creating an entry that allows the user to manipulate its content and/or style. Consider integrating HTML elements such as range sliders, buttons, text inputs, etc. For a full list of the different HTML inputs available, check out this W3 resource. As always, think about how your design decisions relate the theme of your collection.

    To get full credit, make sure you hit all the requirements in the checklist below.

    1. I submitted a link to my entry by 11/26 at 7 PM ET.

    2. I worked in the index.html, style.css, and script.js file of my entry-10 folder.

    3. I used JavaScript to allow the user to manipulate my content and/or style of my entry.

  • Harmonic Collection: Final Presentation Draft

    Create a three-minute presentation that showcases your Harmonic Collection using Figma. Each slide (or “frame”) should be 1920 x 1080 pixels wide. You are welcome to work off this template, but be sure to duplicate the file and make it your own. Your file should be in the [Harmonic Collection] Final Presentation folder. Your presentation should include:

    1. An overview of your collection’s theme.
    2. An overview of how your design decisions relate to the theme of your collection.
    3. A demonstration of your live site. Specifically, show us:
      1. How someone navigates the website.
      2. A deep dive of two entries that you think best represents the overall spirit of the collection. What is each entry about? How does each entry’s design relate to its content?
    4. Reflections on your work, which could include but are not limited to: ways you could extend the project beyond this class, technical pitfalls or achievements, or your favorite part about building your collection.

    To get full credit, make sure you hit all the requirements in the checklist below.

    1. I completed my presentation draft by 11/26 at 7 PM ET.

    2. My presentation draft roughly outlines all of the points above.

    3. My file is in the [Harmonic Collection] 3 Ideas folder within the shared class team on Figma.
