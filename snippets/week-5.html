  • Harmonic Collection: Entry #4

    Create your fourth entry for your Harmonic Collection. For this week, you should:

    1. Start in Figma. Work in the same design file that you have in this project space and work through your ideas. Make sure your name is in your file. Consider typography and layout: how can you break the top-to-bottom, left-to-right default layout of the page?
    2. Go to code. Create an index.html file and style.css file in your entry-4 folder. Translate your design in Figma to a live website using HTML and CSS. Consider how you can break the top-to-bottom, left-to-right default flow of the webpage by utilizing the position property. Additionally, based on what you've learned about nested hovering and targeting, consider how you can make your entry interactive. Can your interactions intentionally speak to your content and overall theme?

    Submit a link to your entry on Canvas for completion.

    2. I worked in the index.html file and style.css file of my entry-4 folder.

    3. I used the position property to create a dynamic layout.

    4. I used nested targeting and hovering throughout my entry to create an interesting interactive experience.
