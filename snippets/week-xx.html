  • HTML Wiki

    Let’s practice some HTML and web typography fundamentals by typesetting a Wikipedia article ourselves! First, choose an existing Wikipedia article you’re interested in working with (e.g. Möbius strip). You’ll then use only HTML to typeset that article.

    Part 1: HTML as Semantics

    Create a new subfolder called exercise-1a within your username.github.io folder. Create a new index.html within that subfolder to work in. Take a look at your chosen article and note the different levels of hierarchy you’ll need. Use semantic HTML such as headings (h1 through h6), ordered lists (ol), and/or unordered lists (ul) to create hierarchy and structure for your content.

    Part 2: HTML as Design

    Now create a new subfolder called exercise-1b within your username.github.io folder. Create a new index.html within that subfolder to work in. Using the same article, use only one HTML element to design it. See how far you can push your design using just that one HTML element: how can hierarchy still be achieved? Can the element be used illustratively? Can the HTML element relate to the contents of the article in some way? Get experimental, get weird, get creative!

    For both exercises, please include a link to the original article you’ve chosen as an HTML comment.

    6. My HTML files for both exercises are named index.html.
