  • Harmonic Collection: Entry #3

    Create your third entry for your Harmonic Collection. For this week, you should:

    2. Go to code. Create an index.html file and style.css file in your entry-3 folder. Translate your design in Figma to a live website using HTML and CSS. For your layout, consider the CSS Box Model and the position properties you learned this week.

    Submit a link to your entry on Canvas for completion.

    To get full credit, make sure you hit all the requirements in the checklist below.

    2. I worked in the index.html file and style.css file of my entry-3 folder.
